Web Utility Suite

A collection of high-performance, client-side web tools built for precision and privacy. No data ever leaves the browser.

🛠 Available Tools

1. PDF Architect (`index.html`)
A professional-grade utility to convert multiple images (JPG/PNG) into a single, formatted A4 PDF document.
* **Logic:** Uses `jsPDF` with mathematical centering and A4 scaling.
* **Privacy:** 100% serverless processing.

2. Life Chronometer (`age.html`)
An aesthetic, high-precision age engine that calculates existence down to the second.
* **Logic:** Real-time entropy tracking using a `365.25` day-per-year leap-year correction.
* **Experience:** Interactive UI with Glassmorphism design and `Canvas-Confetti` success states.

Deployment
This suite is live via GitHub Pages. Access the tools by appending the filename to the base URL:
* `/index.html` -> Image to PDF
* `/age.html` -> Age Calculator
